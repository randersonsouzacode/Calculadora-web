# 🧮 Neon Calculator

Uma calculadora web funcional, responsiva e com um design moderno em *dark mode* (estilo neon). Projetada para oferecer uma interface limpa e intuitiva, focada na experiência do usuário.

## 🚀 Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de praticar a **lógica de programação em JavaScript** e a manipulação dinâmica do DOM. A calculadora executa operações matemáticas básicas de forma eficiente, processando entradas de usuário através de uma interface estilizada.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica da calculadora.
- **CSS3:** Estilização responsiva, efeito neon e design moderno.
- **JavaScript (ES6+):** Lógica de cálculo, tratamento de eventos de clique e atualização do display.

---

## 💡 Principais Funcionalidades
- **Operações Básicas:** Soma, subtração, multiplicação e divisão.
- **Limpeza (Clear):** Botão "C" para resetar o display instantaneamente.
- **Feedback Visual:** Design responsivo com bordas destacadas.
